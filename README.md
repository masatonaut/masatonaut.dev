# masatonaut.dev

Personal portfolio & business site.
Home of the **choimo** micro-tool series.

Built with Next.js 15 + Tailwind CSS v4 + Kanso design system.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Design System**: Kanso
- **Deployment**: Vercel
